Hero Template

The Hero was editorially curated by the Livefyre Marketing team Content was submitted via Comment widget in a fancy HTML structure such that metadata could be parsed out to populate the Hero parts:

  • URL of link
  • Image URL for background image
  • Source of the article
  • Headline to display on Hero
  • snippet of text to preview

    function _getHeroDataFromParagraphs ($bodyHtml) {
